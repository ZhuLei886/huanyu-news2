# 寰宇新闻--HuanYuNews

**寰宇新闻是一个仿今日头条的Web端项目。使用SpringBoot+Mybatis+thymeleaf开发。前端使用bootstrap、html5、css3动画、jQuery、WangEditor富文本编辑器、Webuoloader和Base64处理图片上传、ajax异步处理表单数据,包括后台管理系统。**

## 系统模块

系统功能模块组成如下所示

```
├─管理端
   │  ├─权限管理
   │  ├─新闻管理
   │  ├─日志管理
   │  └─素材管理
├─用户端
   |  ├─首页
   |  ├─新闻展示
   |  ├─新闻推荐
```
